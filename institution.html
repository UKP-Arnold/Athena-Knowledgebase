Präfix

/institutions

Input:

Keiner.

Output:

Gibt die kompletten Daten aller Institutionen zurück.

Spezifische Anfragen

/byInstitutionID/{value}

Input:
"value":

Beschreibt die datenbankinterne, einzigartige ID einer Institution. Diese muss eine Zahl größer gleich 1 sein, ohne führende Null.
Beispiel: 5 oder 42

Output:

Gibt die kompletten Daten der Institution zurück, die die ID hat, die beim Aufrufen des APIs angegeben wurde.

/byName/{value}

Input:
"value":

Beschreibt den Namen einer Institution. Dieser muss exakt sein, Leerzeichen und Sonderzeichen sind erlaubt.

Output:

Gibt die kompletten Daten der Institution zurück, die den Namen hat, der beim Aufrufen des APIs angegeben wurde.

/byPerson/{value}

Input:
"value":

Beschreibt den Namen einer Person. Dieser muss exakt sein, Leerzeichen und Sonderzeichen sind erlaubt.

Output:

Gibt die kompletten Daten aller Institutionen zurück, bei denen die Person, deren Name beim Aufrufen des APIs angegeben wurde, tätig ist.
