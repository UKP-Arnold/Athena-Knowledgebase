A conversational Agent

Um auf die in der Datenbank gespeicherten Daten zugreifen zu können, kann das von uns bereitgestellte REST API verwendet werden. Es ist möglich, alle unten aufgelisteten Entitäten anhand der vorgeschriebenen Parameter abzufragen. Auf das REST API kann unter diesem Link zugegriffen werden.

Die Wikiseite einer jeden Entität nennt ein Präfix, was bei jeder Abfrage über diese Entität an den vorher genannten Link angehängt wird. Weiterhin sind auf jeder Wikiseite alle möglichen Abfragen zu dieser Entität aufgelistet. Diese sind dem Link-Präfix-Konstrukt hinten anzufügen. Eine Anfrage hat möglicherweise Parameter, die vom Nutzer hinzugegeben werden müssen. Diese Parameter sind mit {} umkreist, beim Aufrufen des APIS komplett zu ersetzen, und im dort folgenden Text näher erläutert.

Beispiel einer Abfrage anhand einer Paper ID: https://example.org/papers/byPaperID/10

Durchgestrichene Anfragen sind geplant, oder funktionieren noch nicht.
